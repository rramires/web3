# ERC-20 Sample
ERC-20 Token Project Example

## Packages
```shell
npm init -y 

npm i dotenv  

npm i -D hardhat

npx hardhat init

npm i -D @nomicfoundation/hardhat-verify 
```

## Run and Tests

```shell
# Compile
npm run compile 

# Test 
npm test

# Deploy 
npm run deploy 
```
## Deploy

```shell
# Localhost
npm run deploy

# Sepolia via Infura
npm run deploy_testnet   
```
